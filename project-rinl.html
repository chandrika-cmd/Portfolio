Gate Pass Management System

Spring Boot MySQL Hibernate Bootstrap

Project Overview

Developed during my internship at Rashtriya Ispat Nigam Limited (RINL / Vizag Steel Plant), this enterprise web application automates the issuance and verification of gate passes for visitors, vehicles, and materials.

Key Features:

  • Role-Based Access Control: Secure login for Admins, Security Personnel, and Department Heads.
  • Digital Pass Generation: Instantly generates printable/digital passes with unique QR codes.
  • Approval Workflow: Multi-level approval process for sensitive material movement.
  • Audit Logs: Tracks entry and exit times for enhanced security monitoring.
  • Reporting: Generates daily and monthly visitor reports for management.

Impact

The system significantly reduced the waiting time at the main gate by digitizing the manual lookup process and provided the security team with real-time data on who is currently inside the plant premises.
