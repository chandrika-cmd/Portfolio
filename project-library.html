Library Management System

Java MySQL JDBC Swing/JavaFX

Project Overview

The Library Management System is a robust desktop application designed to streamline the daily operations of a library. It replaces manual record-keeping with an efficient, automated digital system.

Key Features:

  • Book Management: Add, update, and remove book records easily.
  • Member Management: Track member details and borrowing history.
  • Issue & Return: Automated tracking of due dates and fine calculation for late returns.
  • Search Functionality: Quick search by title, author, or ISBN.
  • Admin Dashboard: Comprehensive view of total books, issued books, and active members.

Technical Challenges

One of the main challenges was ensuring data consistency between the Java application and the MySQL database, especially during concurrent transactions (e.g., two admins trying to issue the same book). This was resolved by implementing proper transaction management and locking mechanisms in the database.
